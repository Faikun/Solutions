# WorldSkills Kazakhstan 2025 — Module A (Simple Version)

Это упрощённая версия проекта для начинающих разработчиков.

Главная идея этой версии:
- сохранить требования техзадания;
- сделать код более понятным;
- использовать меньше сложной динамической генерации;
- оставить простую структуру файлов.

## Страницы проекта

- `index.html` — Home
- `seller.html` — For Seller
- `seller-services.html` — For Seller / Paid Services
- `buyer.html` — For Buyer
- `contacts.html` — Contacts
- `policy.html` — Policy

## Как запустить проект

### Вариант 1. Через VS Code + Live Server
1. Откройте папку проекта в VS Code.
2. Установите расширение **Live Server**.
3. Откройте файл `index.html`.
4. Нажмите **Go Live**.

### Вариант 2. Через Python
В корневой папке проекта выполните команду:

```bash
python -m http.server 8000
```

После этого откройте в браузере:

```text
http://localhost:8000
```

## Как изменить ссылки на кнопках `Опубликовать` и `Посмотреть`

Чтобы не менять ссылки вручную на каждой странице, они вынесены в файл `js/config.js`.

Откройте файл `js/config.js` и найдите такие строки:

```js
window.APP_CONFIG = {
  publishLink: 'seller.html',
  viewLink: 'buyer.html'
};
```

### Что означает каждая строка
- `publishLink` — ссылка для всех кнопок `Publish` и `Опубликовать`
- `viewLink` — ссылка для всех кнопок `View` и `Посмотреть`

### Пример изменения
Если нужно отправлять пользователя на другие страницы, замените значения:

```js
window.APP_CONFIG = {
  publishLink: 'https://example.com/publish',
  viewLink: 'https://example.com/catalog'
};
```

После сохранения файла новые ссылки автоматически применятся на всех страницах.

## Как устроен JavaScript в этой версии

В `main.js` оставлены только самые понятные задачи:
- подставить ссылки для кнопок;
- подсветить активный пункт меню;
- обновить стоимость платных услуг по `range`;
- показать сообщение после отправки формы;
- скрыть битые изображения и показать текстовую заглушку.

## Как работают платные услуги на странице `seller-services.html`

У каждой услуги есть `input type="range"`.
Ползунок переключает заранее заданные варианты:
- 1 день
- 2 дня
- 10 дней
- 30 дней

Цены лежат в `js/config.js`, поэтому их легко менять в одном месте.

## Важно про медиа-файлы

В проекте оставлены корректные пути к медиа, как в задании:
- `media-files/logo.png`
- `media-files/map.png`

Если папки `media-files` рядом с проектом нет, сайт всё равно останется рабочим: вместо изображения покажется простая текстовая заглушка.

Если позже вы добавите папку `media-files`, изображения начнут отображаться автоматически.
